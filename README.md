# FinMitra — Cloud-Powered Expense Tracker for Modern India

FinMitra is a cloud-based expense tracking web app that helps users log, visualize, and analyze their daily spending seamlessly.  
Built using React and Firebase, it offers real-time data storage, smart analytics, and an intuitive dashboard for modern Indian users.

---

## Features

- Daily Expense Logging — Add, edit, and delete expenses effortlessly  
- Cloud Sync with Firebase — Access your data from anywhere  
- Secure Authentication — Firebase Email/Password login system  
- Auto-Generated Charts — Monthly spending summaries powered by Recharts  
- Smart Suggestions — Personalized saving tips (planned)  
- Landing Page — A welcoming homepage for first-time visitors  
- Responsive Design — Works perfectly on desktop, tablet, and mobile  
- Deployed on Firebase Hosting — Fast and secure access anytime  

---

## Tech Stack

| Layer | Technology | Purpose |
|-------|-------------|----------|
| Frontend | React (Vite) | Fast and modular UI |
| Styling | TailwindCSS | Responsive, clean design |
| Charts | Recharts | Interactive expense analytics |
| Routing | React Router DOM | Smooth navigation between pages |
| Icons | Lucide React | Lightweight, elegant icons |
| Backend | Firebase Firestore | Real-time NoSQL database |
| Authentication | Firebase Authentication | Secure login/signup |
| Hosting | Firebase Hosting | Simple and fast deployment |
| State Management | React Context API | Global data sharing |

---

## Cloud Service Model

FinMitra is a Software-as-a-Service (SaaS) product built using Firebase (a PaaS), running on Google Cloud (IaaS).

- SaaS: FinMitra — the end-user web application  
- PaaS: Firebase — backend services and deployment  
- IaaS: Google Cloud — underlying infrastructure
